设定每日学习目标
问问自己每天希望在课程中完成什么。设定一个明确的目标可以帮助您保持动力，战胜拖延症。目标应具体且易于衡量，例如 "我将观看模块 2 中的所有视频并完成第一个编程作业"。当你朝着目标前进时，别忘了奖励自己！

创建专门的学习空间
如果你在最初学习信息的地方，就会更容易回忆起这些信息，因此，在家里有一个专门的空间来学习在线课程，会让你的学习更有效。清除空间中的任何干扰因素，如果可能，将其与床或沙发分开。明确区分你学习和休息的地方可以帮助你集中注意力。

提示：您可以将 Coursera 课程的截止日期添加到谷歌日历、苹果日历或其他日历应用程序中。

HTML
Recall HTML 用于描述网页的结构。开发人员使用带有开头和结尾标记的 HTML 元素来 "标记 "HTML 文档。

这些元素构成了网页的结构，并描述了向网页浏览器显示的内容。说到 HTML，了解以下内容很重要：

HTML 在网页浏览器中的作用、

HTML 标记的使用和正确的语法、

以及如何在网页文档中使用 HTML 元素。

在谈论 HTML 时，另一个需要了解的重要概念是文档对象模型（Document Object Model，简称 DOM）。

用户需要能够与网页上的元素进行交互。这意味着 HTML 文档必须以一种 JavaScript 代码可以查询和更新的方式来表示。这就是 DOM 的功能。它是 HTML 文件中对象的模型。

网络开发人员通过 JavaScript 与 DOM 进行交互，以更新内容、设置事件并为 HTML 元素制作动画。
